Адаптивная одностраничная статья о концентрации внимания с тремя темами оформления: светлой, тёмной и автоматической (на основе системных настроек пользователя). Проект выполнен в неоновом стиле с акцентом на типографику и декоративные элементы.

## 📄 Содержание страницы

- **Header** — заглавие "no focus", подзаголовок с описанием темы, меню переключения темы
- **Почему сосредоточиться так сложно** — про многозадачность и дофамин
- **Что снижает концентрацию** — многозадачность, еда, гаджеты
- **Как концентрироваться лучше** — 5 практических советов
- **Галерея изображений** — адаптивная сетка с картинками в стиле "можно в картинках?"
- **Footer** — заглавие "focus" с декоративными элементами

## 🛠 Стек технологий

- **Вёрстка:** HTML5, CSS3 (Grid, Flexbox, Custom Properties)
- **Шрифты:** IBM Plex Mono (Regular, Bold) в формате WOFF
- **JavaScript:** Vanilla JS (ES6+) — переключение тем, сохранение в localStorage
- **Изображения:** PNG с ленивой загрузкой (`loading="lazy"`)
- **Оптимизация:** CSS-переменные, `clamp()` для адаптивной типографики, `prefers-color-scheme`

## ✨ Ключевые особенности

### 🎨 Интерфейс

- **Три темы оформления** — светлая ("День"), тёмная ("Неон"), автоматическая ("Авто")
- **Переключение через классы** — `theme-light`, `theme-dark` на `<html>` с переопределением CSS-переменных
- **Сохранение выбора** — тема запоминается в `localStorage` и восстанавливается при следующем визите
- **Адаптивная типографика** — плавное изменение размеров шрифтов через `clamp()`
- **Мозаичная галерея** — CSS Grid с `grid-column` и `grid-row` для сложной раскладки изображений
- **Декоративные уголки** — абсолютное позиционирование в стиле "decorated zone"
- **Индикатор REC** — появляется только в тёмной теме через CSS-переменную `--display-decorated-element`
- **Неоновые тени текста** — `text-shadow` для заголовков
- **Эффект параллакса** — `background-attachment: fixed` для фона

### ⚡ Производительность

- **Ленивая загрузка изображений** — `loading="lazy"` для всех картинок в галерее
- **CSS-переменные** — единый источник цветов для всех тем
- **Разделение CSS** — `variables.css`, `globals.css`, `style.css`, `dark.css`, `light.css`
- **`font-display: swap`** — избегаем невидимого текста при загрузке шрифтов
- **`prefers-color-scheme`** — автоматическое определение темы по системным настройкам

### ♿ Доступность (a11y)

- **Семантическая вёрстка** — `<header>`, `<nav>`, `<main>`, `<footer>`, `<article>`, `<aside>`
- **Корректная работа с клавиатуры** — стили `focus-visible` для клавиатурной навигации
- **ARIA-атрибуты** — `aria-hidden="true"` для декоративных элементов
- **Управление состоянием** — активная кнопка темы получает `disabled` для предотвращения повторных кликов
- **Атрибуты `alt`** — у всех изображений

### 📐 Адаптивная вёрстка

Три брейкпоинта с использованием современных CSS-техник:

- **Мобильные (< 768px)** — одноколоночная раскладка
- **Планшеты (≥ 768px)** — двухколоночный CSS Grid для секций, мозаичная галерея
- **Десктоп (≥ 1024px)** — горизонтальное меню тем, расширенные отступы

## 🧠 Архитектурные решения

### Разделение CSS по темам

Каждая тема вынесена в отдельный файл (`dark.css`, `light.css`) и переопределяет CSS-переменные из `variables.css`. Это позволяет:

- Легко добавлять новые темы
- Избежать дублирования кода
- Чётко видеть, какие цвета относятся к какой теме

### Механизм переключения тем

1. При загрузке страницы `initTheme()` проверяет `localStorage` на наличие сохранённой темы
2. Если тема не найдена — используется системная настройка через `prefers-color-scheme`
3. При клике на кнопку темы вызывается `setTheme()`, который:
   - Очищает все классы у `<html>`
   - Добавляет класс `theme-{name}`
   - Сохраняет выбор в `localStorage`
4. Активная кнопка получает класс `_active` и атрибут `disabled`

### CSS Grid для галереи

Мозаичная раскладка изображений реализована через CSS Grid с `grid-column` и `grid-row`:

```css
.sunset {
  grid-column: 1 / span 2;
  grid-row: 1 / auto;
}
.ice-cream {
  grid-column: 3 / auto;
  grid-row: 1 / span 2;
}
.tape {
  grid-column: 1 / auto;
  grid-row: 2 / span 2;
}
```
